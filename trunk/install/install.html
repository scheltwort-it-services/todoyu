Todoyu Installation

  1. Upload the todoyu folder to your server into your prefered directory (ex: www.yourdomain.com/todoyu)
  2. Create a new mysql database for todoyu (todoyu will delete existing todoyu tables in this database!)
  3. Download and add external tools which are needed by todoyu
    1. TinyMCE
      http://tinymce.moxiecode.com/download.php (Main package)
      Copy the "tiny_mce" folder into the todoyu/lib/js/ folder
    2. phpMailer
      http://sourceforge.net/projects/phpmailer/
      Copy the content of the archive into todoyu/lib/php/phpmailer
    3. jsCalendar
      http://www.dynarch.com/static/jscalendar-1.0.zip
      Copy the content of the archiv info todoyu/lib/js/jscalendar
  4. Call www.yourdomain.com/todoyu/install for the finishing steps
