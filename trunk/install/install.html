Todoyu Installation

Info for SVN users: You only have to do step 3 and 5!

  1. Download the latest version from Sourceforge (http://sourceforge.net/projects/todoyu/)
  2. Extract the archive and upload the todoyu folder into the document root of your webserver. It has to be accessable for you browser (ex: www.example.com/todoyu)
  3. Create a new MySQL database for todoyu (the installer will remove existing todoyu tables in this database)
  4. Download and add external tools which are needed by todoyu
    1. TinyMCE
      http://tinymce.moxiecode.com/download.php (Main package)
      Copy the "tiny_mce" folder into the todoyu/lib/js/ folder
    2. phpMailer
      http://sourceforge.net/projects/phpmailer/
      Copy the content of the archive into todoyu/lib/php/phpmailer
    3. jsCalendar
      http://www.dynarch.com/static/jscalendar-1.0.zip
      Copy the content of the archiv info todoyu/lib/js/jscalendar
  5. Call www.example.com/todoyu/install to start the installer and finish the installtion (Start installer: /install)
